# Getting Started

***Flutter SDKのインストール：***

https://flutter.dev/docs/get-started/install/macos

***Flutter プロジェクトの作成：***

今回はネイティブ言語をswiftとkotlinに設定
```
$ flutter create -i swift -a kotlin --with-driver-test flutter_sample
```

***起動：***

```
$ cd flutter_sample
$ flutter run
```
